# flink-drools-dynamic-pattern

## env 

- WSL
- openjdk 17
- apache flink 1.18.0
- Drools 8.44.0.Final

## how to use

- execute netcat command
```
nc -lk 9999
```

- execute main function 

- in your netcat terminal, type values as bellow and hit return 
```
1,80
2,90
3,95
```
